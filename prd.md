| 순서 | 강의명 | 시간 | 가격 | 핵심 학습 내용 |

- STEP 1, 2 완료 후 → "이제 진짜 서비스 만들 준비 됐어요!" 배너
- 수강 후기 강조: "2주 만에 5개 웹 배포" 실제 사례

## 기술 구현 - Supabase 스키마

### 테이블 설계

```sql
-- 로드맵 트랙
CREATE TABLE tracks (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  name TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL, -- 'ai-beginner', 'ai-developer', 'spring-backend'
  tagline TEXT, -- "코딩 몰라도 4주 만에 수익화 서비스 런칭"
  description TEXT,
  target_audience TEXT,
  badge TEXT, -- '인기 트랙', '개발자 필수', '취업 완성'
  estimated_duration TEXT,
  sort_order INT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 강의 정보
CREATE TABLE courses (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  track_id UUID REFERENCES tracks(id),
  title TEXT NOT NULL,
  slug TEXT UNIQUE NOT NULL,
  level TEXT, -- 'Lv0', 'Lv1', 'Lv2', 'Lv3', 'Premium'
  duration TEXT,
  price INT,
  sale_price INT,
  inflearn_url TEXT,
  thumbnail_url TEXT,
  status TEXT DEFAULT 'published', -- 'published', 'coming_soon'
  expected_release TEXT, -- '2025년 Q2'
  sort_order INT,
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- 출시 알림 신청 (리드 수집)
CREATE TABLE course_waitlist (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  course_id UUID REFERENCES courses(id),
  email TEXT NOT NULL,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  UNIQUE(course_id, email)
);

-- 트랙 퀴즈 응답 (분석용)
CREATE TABLE quiz_responses (
  id UUID DEFAULT gen_random_uuid() PRIMARY KEY,
  coding_experience TEXT, -- 'none', 'some', 'developer'
  goal TEXT, -- 'side-project', 'job', 'productivity'
  recommended_track_id UUID REFERENCES tracks(id),
  created_at TIMESTAMPTZ DEFAULT NOW()
);

-- RLS 정책
ALTER TABLE tracks ENABLE ROW LEVEL SECURITY;
ALTER TABLE courses ENABLE ROW LEVEL SECURITY;
ALTER TABLE course_waitlist ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Public read" ON tracks FOR SELECT USING (true);
CREATE POLICY "Public read" ON courses FOR SELECT USING (true);
CREATE POLICY "Anyone can subscribe" ON course_waitlist 
  FOR INSERT WITH CHECK (true);
```

## 개발 로드맵

### Phase 1: MVP (2주)
- [ ] Supabase 테이블 생성 및 초기 데이터 입력
- [ ] 로드맵 메인 페이지 (3개 트랙 카드)
